Data & SQL

8 June 2021

I decided to enrol in a final course, Data and SQL, as Code First Girls has been a fantastic opportunity to learn and I wanted to add another programming language into my CV. Plus, I've been enjoying working on projects with girls and the courses have allowed me to gain confidence and soft skills.

We used MySQLWorkbench as the IDE.

Final Project

This SQL course was not easy like Web Development and Python but I was able to do the required tasks for final project with a coursemate and use the slides and W3Schools as guides. It is true that it takes some practice to learn and understand what each code does. So it is important to be patient and being confused is a productive part of the learning process!

Initially, I wanted to create a database on films but my coursemate suggested a gym management database and I thought it would be a good idea to do a sort of a real life database. All names, emails, phone numbers and addresses in the tables are random and not based on real people. We created eight tables and completed the requirement tasks. We tested the queries to see they work and are showing correct information.

During the presentation, we showed our project to the class and two of the queries that were interesting to show were the stored procedure where you can update users' table if a user has updated their password and creating an event where a table is created that stores the total monthly revenue of the gym.

Challenges included: using hyphens in tables which led to altering and updating the tables and creating an event.

I felt bittersweet after finishing all of Code First Girls courses and I'm hoping to do a nanodegree in software engineering with them. I'm also excited to become a volunteer instructor to pay back to Code First Girls as they helped me find a new passion during the pandemic and I continued to use the skills that I learnt from the courses into my personal projects. I want to inspire the female/non-binary identifying young people to get into tech and help them get their start in learning to code.

about
