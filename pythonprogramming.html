Python Programming

25 March 2021

After completing Introduction to Web Development, I decided to expand my knowledge and skills. I applied Python Programming and prior to starting the course, I tried BrightNetwork Technology Academy: Starting To Code In Python to give me an idea of how Python works. I believe that Python was much easier to understand than Javascript.

We used PyCharm as the IDE.

Final Project

I had a lot of fun creating this project as I like Harry Potter and playing Top Trumps. But creating it was not as straightforward as we thought. We encountered few problems that we were not sure how to solve but fortunately, we were able to use Google and if the problem wasn't solved, the instructors were kind enough to help us.

We were given three project briefs to choose and we chose Top Trumps Harry Potter. After completing the required tasks and extended tasks, we added our own ideas such as allowing the user to type how many rounds of the game they want to play and if they win or lose, a turtle graphics is shown in a new window. I changed the colours of the text to allow the user to read easier and it would be appealing. The statistics of each character were imported from a CSV file. The user is given five randomly generated characters to choose and they play against the computer. The scores are counted and shown at the end of the game. They are also written to another CSV file for the user to read the scores.

Few hours before the last session, as a group we decided to have a Slack video chat to prepare for the presentation and we added comments to allow us remember and explain the functions. The presentation was successful and we received a certificate for project excellence after my coursemates voted.

Challenges included: randomly picking a row and making the function return the stats of that character and completing the extended tasks e.g. recording the outcome of each round and total scores in a file, fixing the wrong result and choosing a character from five randomly choices.
