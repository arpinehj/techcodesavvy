Web Development

24 November 2020

During the COVID-19 pandemic in 2020, I decided to look for more information about Code First Girls and try one of their courses after completing BrightNetwork Internship Experience Technology. My first course was Introduction to Web Development and after applying, I received materials such as a guide and presentation documents to get ready for the course.

We used Visual Studio Code as the IDE. It is very easy to download and it is free! I also use it for personal projects!

Description
"An introduction to web development for budding coders, and those who need somewhere to get started! Learn the fundamental skills of front-end web dev, including three dedicated sessions on JavaScript. By the end of the course, you will be able to build a website from scratch!"

Sessions
  • HTML basics
  • CSS basics
  • Recap & Project Design
  • Javascript Part 1 (Overview, data types)
  • Javascript Part 2 (Loops, functions and scope)
  • Javascript Part 3 (Objects and the DOM)
  • Github, Frameworks & Project work
  • Project presentations & Careers in Web Dev
Final Project

In week 3, we were randomly assigned into groups of four and we started to think of different ideas that we wanted to create a website. I suggested to create a website that is meaningful and impactful so we decided to focus on plastic ocean. There was a criteria in order to have a project success: planning, communication, skills, tools, process, management, teamwork. We also had two lists of 'must have' and 'nice to have' to follow and made sure to use the skills and tools that we learnt into practice in order to receive a certificate.

We communicated and planned the website by assigning us tasks such as creating wireframes and pages and discussing on web design. I created the wireframes and suggested each page's topic such as news and get involve with charities. The wireframes helped us to create the HTML and CSS and we added some Javascript such as slideshow and scroll to top button to allow user interactivity. By the end of the course, only two, including myself, were working on the project.

Finally, we organised our notes for our presentation and allowed our coursemates to ask us questions. They gave us their thoughts about the website and they all liked it! We completed all of the points except one which is the website responsiveness in the criteria due to the timing. Therefore, we didn't receive a certificate for project excellence but received one for completing the course and project on time!

After finishing the course, I spent some time getting the website responsive!

Challenges included: getting the website responsive and putting text on image.
